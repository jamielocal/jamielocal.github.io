Setup

To Setup UA Library Do This

  1. Download Using Pypi (PIP) Or Direct From Github
  2. Install Using pip install ua_lib.
  3. Or git clone https://github.com/jamielocal/UALib | cd UALib | python3 -m pip install -U . To install From Github

Database

  • UADatabase('database.ua')

    Opens a Line To That Database File. Followed By connect_db.

  • connect_db()

    Connects The Database From UADatabase. Disconnection Needed At the End of the Program.

  • disconnect_db()

    Disconnects The Database From UADatabase.

  • add_entry('Value', {'value': 'value'})

    Creates a Value in the Connected Database!

  • get_entry('Value')

    Gets an Entry Based Off of First Value (Key).

  • delete_entry('Value')

    Deletes an Entry Based off of First Value (key).

  • update_entry('Value', {'value': 'Name'})

    Updates a Value in the Connected Database.

  • get_all()

    Gets all Database Entries.

  • create_database('database.hdb')

    Creates a Database (Either .hdb Or .ua).

  • convert_sqlite('sqlitedatabase.db', 'converted.ua')

    Converts a SQLITE3 Database To a UADatabase.

Examples can be found at https://jamielocal.github.io/examples/db
